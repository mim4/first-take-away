In this first take away we have created a Shiny App. This Shiny App shows a summary of the dataset Wine (obtained from kaggle). The chosen data set contains more than 6000 observations of different wines and its main properties, such as alcohol, pH, density, etc. This data set contains a variable called quality, which gives a score between 0 and 10 to every wine.

In the shiny App we have the following tabs:
  • Data: in this tab we can select what to show in the main panel. By defect, it shows an introduction to the dataset. We can also see a summary of our data (which was created using the function dfSummary), and we can also see a table where all the raw data is.
  • Machine Learning: this tab was inspired by the following App. Here we can do a classification on the variable type of the dataset. This variable can take value 'white' or 'red'. To carry out the classification we can select the random seed, the proportion of data that will be the train and finally the method that will do the classification. We can select between lda, qda and random forest. After selecting the parameters we can see the result in two tabs. The first one which shows the fit of the model and the second one in which we can see the confusion matrix and the general results. Also, we can click on generate pdf and a pdf of classification analysis will be created using the parameters selected.
  • Plots of the data: here we can select what kind of plot we want to see (histogram, boxplot or scatterplot). The histogram is in relationship with type, as well as the boxplot. In the scatterplot, we have the regression line for a linear model that fits the two selected variables. Also, we can click on a certain point in the graph and up to 10 observations will be shown in the table below (this 10 observations are the most close to the clicked site with a threshold of 10).
  • Interactive plots: here we can see a pieplot of variable type, a 3d plot of variables alcohol, total sulfur dioxide and free sulfur dioxide and finally a buble chart where we can see the variables alcohol and total sulphur dioxide, with type and quality too.
